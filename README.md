# This is my simple project 

Requirements:
- Python 3.11.9
- poetry 1.8.3

- ...

# For run this project
1. First we need install correct version of python
```shell
pyenv install 3.11.9
```
2. Next install poetry env
```shell
poetry env use python
```
3. Activate env
```shell
poetry shell
```
4. Install requirements
```shell
poetry install
```
5. ... and
```shell
python Main.py
```
